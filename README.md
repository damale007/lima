# LIMA 1.0: Framework Ligero de PHP para Aplicaciones Web en MVC

## Descripción del Proyecto

**LIMA 1.0** es un **Framework de PHP** diseñado para la **realización de aplicaciones web bajo el patrón de diseño MVC** (Modelo-Vista-Controlador).

Es un *framework* destacado por ser **muy ligero y fácil de usar**.

## Características Principales

*   **MVC Nativo:** Permite construir aplicaciones web siguiendo la arquitectura Modelo-Vista-Controlador.
*   **Ultraligero:** Su tamaño es mínimo, ya que **apenas ocupa 400 Kb**, lo cual incluye un editor de comandos.
*   **Generación Automática:** Permite la creación fácil y automática de componentes clave desde la línea de comandos.
    *   Entre los elementos que se pueden crear fácilmente se encuentran: **modelos, controladores, validaciones y formularios**.
*   **Tecnología Híbrida:** Si bien el *framework* principal está escrito en PHP, el editor de comandos asociado fue desarrollado en C#.
    *   *Nota:* El editor de comandos es **solo para Windows**.

## Tecnología Utilizada

| Componente | Lenguaje/Tecnología | Notas |
| :--- | :--- | :--- |
| Framework Principal | **PHP** | Lenguaje base del *framework*. |
| Herramienta de Comandos | **C# sharp** | El editor de comandos está escrito en C#. |
| Compatibilidad del Editor | **Windows** | El editor de comandos es solo para Windows. |

## Uso en Producción (Ejemplo)

Actualmente, el *framework* LIMA está siendo utilizado para adaptar a MVC la **Web de la Hermandad de la Santa Cruz de la Calle Cabo**. La nueva versión estará disponible en abril. Esta web está construida originalmente en HTML5, CSS3, Javascript, PHP y MYSQL.

La web (www.callecabo.com) es un ejemplo de aplicación compleja desarrollada, ya que incluye numerosas funcionalidades, tales como:

*   Sistema de gestión de **noticias, eventos, vídeos y música**.
*   Sistema de **usuarios**.
*   Participación en **Comunidad**, que es una red social propia de la web.
*   Gestiones avanzadas, incluyendo la realización de **pujas, subastas y reserva de entradas**.
*   Capacidad para **apuntarse a Romerías** y realizar **trámites de Secretaría**.
